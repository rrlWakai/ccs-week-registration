## 📂 Installation Guide

1. Clone the repository:

   ```bash
   git clone https://github.com/rrlWakai/ccs-week-registration.git
   ```

2. Move the project to your local server directory (e.g., `wamp/www`)

3. Import the database:

   * Open phpMyAdmin
   * Create a database
   * Import the `.sql` file

4. Configure database connection:

   * Update `/config/db.php` with your credentials

5. Run the project:

   * Start WAMP/XAMPP
   * Open in browser:

     ```
     http://localhost/ccs-registration
     ```
